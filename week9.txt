1.clone git จากที่เคยทำมาทำงานโดยกดลิ้งจาก git hub ได้เลย
  แล้วใช้คำสั่ง git clone ..URL..
2.สร้าง branch จากคำสั่ง git branch feature (สมมุติชื่อบรานช์คือ feature)
3.ดูว่าเราอยู่บรานช์ไหนและมีกี่บรานช์ ด้วยคำสั่ง git branch * จะอยูี่หน้าบรานช์ที่เราอยู่
  branch หลักคือ master
4.เปลี่ยนบรานช์ด้วยคำสั่ง git checkout ....Feature(ชื่อบรานช์)... แล้วจะเห็นว่าวงเล็บด้านหลังเปลี่ยน
  ไปเป็นชื่อบรานช์ที่เราเปลี่ยนไปแล้ว ถ้าไม่ใช่ก็แสดงว่าพิมพ์ผิด ลองอีกที หรือถ้าอยากเช็คให้แน่ใจอีกทีก็เช็คตามข้อ 3
5.เพิ่มไฟล์ ในบรานช์ที่เราสร้าง -> add -> commit
6.กลับมา branch master (จะเห็นว่าไฟล์ที่เราสร้างในบรานช์อีกอันจะไม่มีในนี้) เพิ่มไฟล์ -> add -> commit
7.git log --all --decorate --oneline --graph ดู checkpoint จะยังไม่มีไฟล์นี้ขึ้นที่ github
8.push ขึ้น github ด้วยคำสั่ง push origin master (อยาก push ไฟล์ไหนขึ้นดู branch ให้ตรงกับที่อยากจะ push)
9.ถ้าเราอยากรวมไฟล์ไปไว้ใน master ให้อยู่ใน branch master แล้วใช้คำสั่ง 
  git merge ..(ชื่่อบรานช์) 
  9.1 ถ้ามีข้อความ conflict ให้เลือกว่าจะเอาข้อมูลจาก branch ไหน
  9.2 เมื่อเลือกแล้วให้ merge
10.การ pull คือ เมื่อเรามีไฟล์ทั้งหมดอยู่แล้ว และมีคนแก้ไขไฟล์อยู่ต่างเครื่องกัน 
   เราจึงต้องการเอาข้อมูลหลังจากที่อีกเครื่องแก้แล้วมาด้วย
   ด้วยคำสั่ง git pull origin master ถ้าแก้เพิ่มเสร็จแล้วก็ push
   10.1 ถ้า push แล้วมีข้อความ conflict (แก้ไฟล์เดียกันในต่างเครื่อง) ให้เลือกว่าจะเอาข้อมูลจาก branch ไหน
   10.2 commit แล้ว push ใหม่
